# valo-accounts

Valorantの複数アカウント管理をサポートするデスクトップアプリケーション

## 概要

valo-accounts は、複数の Valorant アカウントを効率的に管理するための専用ツールです。アカウント情報の登録・管理から、ランクとレベルの自動取得、ワンクリックでのアカウント切り替えまで、Valorant プレイヤーの日々の業務を簡素化します。

## 主な機能

### アカウント管理
- **アカウント登録**: RiotID、タグライン、ログインID/メールアドレス、パスワードを登録
- **アカウント一覧表示**: 登録済みのアカウントをランク・RiotID・タグライン付きでリスト表示
- **ランク自動取得**: Web API を通じて現在のアカウントランクとレベルを自動取得・更新

### プロセス管理
- **Valorant 起動状態の追跡**: アプリケーション実行中のプロセス監視
- **Riot Client 起動状態の追跡**: クライアントの実行状態を自動検出
- **Valorant 強制終了**: ボタン操作でプロセスを即座に終了
- **Riot Client 強制終了**: クライアント側も同様にワンクリック終了

### アカウント切り替え
シンボリックリンク機能により、`%LOCALAPPDATA%/Riot Games/Riot Client/Data` のリンク先を動的に変更することで、アカウント間の高速切り替えを実現

## 動作フロー

### 初期セットアップ
1. アプリケーション起動
2. 「アカウント追加」で新規アカウント情報を入力
   - RiotID
   - タグライン
   - ログインID/メールアドレス
   - パスワード

### 日常使用
1. アカウント一覧から使用したいアカウントを選択
2. 「アカウント切り替え」ボタンをクリック
3. Valorant/Riot Client が起動している場合は強制終了
4. アカウント切り替え完了後、自動でクライアントを起動

### ランク更新
1. アカウント一覧の「更新」ボタンをクリック
2. Web API 経由で最新のランク・レベル情報を取得
3. リスト上に自動反映

## システム要件

- **OS**: Windows 10 以降
- **メモリ**: 2GB 以上推奨
- **Valorant**: インストール済み
- **Riot Client**: インストール済み

## インストール方法

1. [GitHub Releases](https://github.com/noma99t/valo-accounts/releases) から最新版の exe ファイルをダウンロード
2. ダウンロードした exe ファイルを実行
3. アプリケーションが起動すれば、インストール完了

## 技術仕様

### 技術スタック
- **フロントエンド**: React 19 + TypeScript
- **デスクトップフレームワーク**: Tauri 2
- **ビルドツール**: Vite
- **バックエンド**: Rust
- **データベース**: SQLite
- **セキュリティ**: パスワード暗号化保存

### データ管理
- アカウント情報は SQLite データベースで一元管理
- パスワードは暗号化された状態で保存
- ローカルストレージのため、プライバシーを保護

## 使用上の注意

- パスワードは安全に暗号化されて保存されます
- アプリケーション削除時は、登録済みアカウント情報も削除されます
- アカウント切り替え時にプロセスキルが行われるため、未保存のゲーム情報は失われる可能性があります
